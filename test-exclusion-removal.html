Testing Exclusion Removal Scenario

This page tests the specific scenario where a URL is already excluded when you visit the page, and then you remove it from the exclusion list.

  2. Navigate to this page - should show disabled state with no conversions

  4. Verify that conversions appear immediately (without refreshing)
