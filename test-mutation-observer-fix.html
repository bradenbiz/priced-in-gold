Testing Mutation Observer Fix

This page tests the fix for the mutation observer that was causing periodic re-conversion of prices even when the URL was excluded.

Test Steps:

  1. Open browser console (F12) to see debug logs
  2. Load this page and verify USD prices are converted to KAU
  3. Click "Add Dynamic Price" button to trigger mutation observer
  4. Verify new prices are converted
  5. Open extension popup and add this page's URL to exclusion list
  6. Click "Add Dynamic Price" button again
  7. Verify NO new conversions happen (should see "skipping processing - URL is excluded")
  8. Remove URL from exclusion list
  9. Click "Add Dynamic Price" button again
  10. Verify conversions resume
